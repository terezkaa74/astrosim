# Quick Start Guide

## For Non-Technical Users

### What You Get

A simple web app that helps you read and understand research papers without needing internet after the first time you open it.

### How to Use It

1. **Open the App**
   - Visit the website link provided
   - Bookmark it for easy access later

2. **Install to Desktop (Optional but Recommended)**
   - Look for an "Install" button in your browser's address bar
   - Click it to add the app to your desktop
   - It will now work like any other program on your computer
   - Works offline after installation

3. **Upload a PDF**
   - Click or drag-and-drop a research paper PDF
   - Wait a few seconds while it processes

4. **Explore Your Document**
   - **Document Tab**: See the paper's structure, sections, and tables
   - **Summary Tab**: Read an automatic summary with main points
   - **Ask Questions Tab**: Type questions and get answers from the paper

5. **Export Information**
   - In the Summary tab, click export buttons to save:
     - .txt file (plain text summary)
     - .md file (formatted markdown)
     - .csv file (tables from the paper)

### Tips

- **Best Results**: Works great with academic/research papers that have clear sections
- **Questions**: Ask simple, specific questions like:
  - "What is the main finding?"
  - "What methods were used?"
  - "What are the results?"
- **Privacy**: Everything happens in your browser. Your PDFs never leave your computer.
- **Offline**: After the first visit, close your internet and it still works!

### What Makes This Different

- No account needed
- No login required
- No internet needed (after first load)
- No data sent to servers
- No API keys or subscriptions
- Completely free

### Troubleshooting

**PDF Won't Upload**
- Make sure it's a .pdf file
- Try a different PDF to test

**No Answer to My Question**
- Rephrase your question
- Make sure the answer is actually in the document
- Try more specific keywords

**App Not Working Offline**
- Visit the site once with internet first
- Make sure you installed it or bookmarked it
- Check that you're using a modern browser (Chrome, Firefox, Edge, Safari)

### Browser Support

Works on:
- Chrome (recommended)
- Firefox
- Microsoft Edge
- Safari (desktop and mobile)

### Mobile Use

Yes! It works on phones and tablets too:
- Visit the site on your mobile browser
- Use "Add to Home Screen" on iOS
- Use "Install App" on Android
- Works offline on mobile as well
